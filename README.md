\# SENTINEL

\*\*Bayesian Multi-Object Tracking with Adaptive Sensor Fusion\*\*



A real-time multi-object tracking system using Bayesian estimation, sensor fusion (LiDAR + Radar), and ML-based noise adaptation for autonomous driving applications.



\## Technologies

\- Python 3.10+

\- nuScenes Dataset

\- NumPy, SciPy, Matplotlib

\- scikit-learn

\- Kalman Filtering



\## Project Structure

```

SENTINEL/

├── data/              # nuScenes dataset

├── src/               # Source code

├── notebooks/         # Jupyter notebooks

├── docs/              # Documentation

└── results/           # Outputs

```
